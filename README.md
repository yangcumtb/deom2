拿到项目包之后：
执行
npm install
(该命令会自动下载所需要的依赖包：
  "dependencies": {
    "cors": "^2.8.5",
    "express": "^4.20.0",
    "jquery": "^3.7.1",
    "ol": "^10.1.0",
    "pg": "^8.12.0",
    "shapefile": "^0.6.6"
  })
  执行完成之后，运行
  npm run dev
  该命令启动地图应用。
  在终端进入/src目录，执行
  node serve.js
  该命令启动数据库连接应用，可以访问数据库
